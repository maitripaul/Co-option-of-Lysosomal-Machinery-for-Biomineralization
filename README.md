# Co-option-of-Lysosomal-Machinery-for-Biomineralization

This repository contains code used to analyze previously published S. lacustris RNASeq data by Musser et al., 2021 (https://www.science.org/doi/full/10.1126/science.abj2949), specifically to pull out expression profiles of genes involved in spiculogenesis
The files for the code are in the data folder
